# iChatGPT
#### 紹介：ChatGPTアプリ無料プランには、メッセージの回数制限、画像生成の利用制限、利用モデルの制限などがあります。Plusの月料金三千円ちょっと高いだと思います、だからこのアプリを作りました。
#### 自分のOpenAIのAPIを使って、これはかなり安いです。
話の内容は文字化ができます、
#### 使い方法：iChatGPTフォルダの中にconfig.xcconfigというファイルを作って、内容は
```CHATGPT_API_KEY = (OpenAI の API keys の SECRET KEY)```
するとアプリが使えます。
